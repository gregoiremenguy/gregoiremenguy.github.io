
I am the lead researcher and main developer of the following program analysis platforms:

* **The [Xyntia](https://github.com/binsec/xyntia) black-box deobfuscator**
    - Description: Xyntia uses program synthesis to simplify local but highly obfuscated code snippets;
    - Linked publications: [CCS 2021](https://dl.acm.org/doi/pdf/10.1145/3460120.3485250), [CCS 2025](https://dl.acm.org/doi/pdf/10.1145/3719027.3765134) 

* **The [PreCA](https://github.com/binsec/preca) black-box precondition learner**
    - Description: PreCA leverages constraint acquisition to perform precondition inference in black-box and with correctness guarantees. 
    - Linked publications: [IJCAI 2022](https://cea.hal.science/cea-04258914/file/2022-ijcai.pdf), [KR 2023](https://proceedings.kr.org/2023/50/kr2023-0050-menguy-et-al.pdf), [JAIR 2025](https://www.jair.org/index.php/jair/article/download/16206/27136)
